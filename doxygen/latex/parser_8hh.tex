\doxysubsection{Detailed Description}
LL Recursive Descent Parser. 

\begin{DoxyAuthor}{Author}
Tony Valentine The file contains the Parser class definition for Tealang\+V1. The parser is implemented as a generic L\+Lk recursive descent parser. The first set for Tealang is unique for all instaces outside of identifiers and function calls, and as such the majority of the implementation code take an L\+L1 approach.
\end{DoxyAuthor}
The implementation is heavily based on that found in the \char`\"{}\+Parsing\char`\"{} chapter of \char`\"{}\+Engineering a Compiler\char`\"{} with all the functions wrapped inside 1 class that generates the A\+ST for the program.
