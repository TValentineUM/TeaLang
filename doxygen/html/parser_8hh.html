Detailed Description

LL Recursive Descent Parser.

Author
Tony Valentine The file contains the Parser class definition for TealangV1. The parser is implemented as a generic LLk recursive descent parser. The first set for Tealang is unique for all instaces outside of identifiers and function calls, and as such the majority of the implementation code take an LL1 approach.

The implementation is heavily based on that found in the "Parsing" chapter of "Engineering a Compiler" with all the functions wrapped inside 1 class that generates the AST for the program.
